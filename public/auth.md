# auth.md

VEPANDO does not currently offer public self-service OAuth/OIDC registration for third-party agents.

Agents can use the public, read-only discovery resources at:

- `https://vepando.com/llms.txt`
- `https://vepando.com/.well-known/api-catalog`
- `https://vepando.com/docs/api`
- `https://vepando.com/.well-known/agent-skills/index.json`

For partnership or integration access, contact `info@vepando.com`. Do not assume token issuance, account creation, or access to protected internal APIs unless VEPANDO provides explicit credentials through a separate agreement.
